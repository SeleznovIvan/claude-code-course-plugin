## What You'll Build

```
your-repo/
├── CLAUDE.md                    # Project context, conventions, agent patterns
├── .mcp.json                    # MCP server configurations
├── .claude/
│   ├── settings.json            # permissions.deny, hooks, security
│   ├── skills/                  # Reusable team standards
│   └── commands/                # Custom slash commands
├── .github/workflows/
│   └── claude-*.yml             # CI/CD with Claude reviews
└── scripts/
    └── claude-*.sh              # Headless automation
```

## Who Is This For?

```bash
$ claude --help roles
```

- `--frontend` — React, Vue, Angular, CSS, component libraries
- `--backend` — APIs, databases, microservices, server-side
- `--qa` — Test automation, E2E, integration testing
- `--devops` — Infrastructure, CI/CD, containers, cloud
- `--data` — Pipelines, analytics, ML, data engineering
- `--fullstack` — Mix of frontend and backend

No prior Claude Code experience needed.

## Quick Start

1. **Install the plugin**
   ```bash
   claude plugin marketplace add https://github.com/SeleznovIvan/claude-code-education
   claude plugin install cc-course@claude-code-education
   ```

2. **Navigate to your project**
   ```bash
   cd /path/to/your/repo && claude
   ```

3. **One-time setup**
   ```
   /cc-course:setup
   ```

4. **Start learning**
   ```
   /cc-course:start 1
   ```

## Powered by Your Session History

Most courses give everyone the same generic exercises. This one reads your actual Claude Code usage and adapts.

The bundled **cclogviewer MCP** server parses your local Claude Code session logs, giving the course instructor (Claude itself) real-time access to how you work:

## For Educators & Teams

This course is designed for both **self-study** and **instructor-led** delivery. Two built-in commands enable a complete homework submission and review workflow:

<div class="cli-block">
<span class="dim">── Student Side ──────────────────────────────────</span><br/>
<span class="prompt">$</span> /cc-course:submit<br/>
<span class="output">Packaging Module 1 submission...</span><br/>
<span class="output">✓</span> Student work files collected<br/>
<span class="output">✓</span> Progress snapshot saved<br/>
<span class="output">✓</span> Session logs exported via MCP<br/>
<span class="output">✓</span> Created: seminar1-jane-doe-2026-03-18.zip<br/>
<br/>
<span class="dim">── Instructor Side ───────────────────────────────</span><br/>
<span class="prompt">$</span> /cc-course:review seminar1-jane-doe-2026-03-18.zip<br/>
<span class="output">Launching reviewer agent...</span><br/>
<span class="output">✓</span> Instructor report: reviews/instructor-report.pdf<br/>
<span class="output">✓</span> Student feedback: reviews/student-feedback.pdf<br/>
<span class="output">✓</span> Score: 87/100 across 5 dimensions<br/>
</div>

**`/cc-course:submit`** — Students package their completed module work into a zip archive containing their CLAUDE.md, commands, skills, hooks, session logs, and progress data. Share the zip with the instructor.

**`/cc-course:review <path>`** — Instructors run a comprehensive AI-powered review of student submissions. Produces detailed feedback across 5 scoring dimensions, with separate reports for the instructor (full analysis) and the student (actionable feedback). Supports single submissions or batch review of an entire class.
